# API Endpoints

## HTML API

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Listings

- `GET /api/university/listings`
  - Listings index/search
- `POST /api/university/listings`
- `GET /api/university/listing/:id`
- `PATCH /api/university/listing/:id`
- `DELETE /api/university/listing/:id`
